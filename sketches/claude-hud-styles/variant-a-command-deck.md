## Variant: Command Deck

### Design stance
Task cards first: the HUD always shows one clearly focused command/task card and a detail pane.

### Key choices
- Layout: left card stack + right detail stream.
- Typography: medium monospace, readable at 480×640.
- Interaction: Up/Down selects cards; Enter opens/acts; Back exits the focused area.
- Permission prompts can become focused cards instead of full-screen surprises.

### Trade-offs
- Strong at: daily usage, task switching, model/permission choices.
- Weak at: raw Claude Code logs; long output needs detail mode.

### Best for
General out-of-home glasses use where the user needs confidence more than maximum text density.
