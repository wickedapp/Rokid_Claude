## Variant: Focus Lanes

### Design stance
Glasses-safe control surface: big focusable lanes make the current mode unmistakable.

### Key choices
- Layout: vertical task / output / action lanes.
- Typography: largest of the three variants.
- Interaction: Up/Down changes lane; Left/Right changes lane options; Enter enters/commits; Back exits lane.
- Designed around accidental input safety.

### Trade-offs
- Strong at: clarity, new-user learnability, low cognitive load.
- Weak at: lower information density; output area is smaller.

### Best for
First production pass if the priority is reliable operation while walking or using limited controls.
